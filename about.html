About

Group - Data Visualisation Project

For the Data Visualisation Project, our team has applied a structured, iterative design process to research, design, and build an interactive data visualisation, displayed on a website.

We used a dataset from the OECD Environment Statistics to provide meaningful insights on environmental challenges.

GitHub Repository

Check out our project on GitHub: Data Visualisation GitHub

Data Source

Dataset used: OECD Plastic Leakage to the Environment by Region

Cleaned Dataset

The cleaned dataset can be accessed here: Cleaned OECD Plastics 2019-2029.csv

GeoJSON Files

To generate the GeoJSON files for the OECD regions, we downloaded individual country maps from Cartography Vectors. These maps were then combined using GeoJSON.io and saved as GeoJSON files.

We created four separate GeoJSON files because large files can slow down the loading time of the website. Splitting the data into smaller chunks improves the performance and ensures faster load times, especially for users with slower internet connections or devices with limited processing power.
